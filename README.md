6. ds:并查集
7. dfs:深度优先查找算法

10. bs:二分查找算法
11. maths:数学
12. ps:前缀和
13. bt:回溯法
14. stack:单调栈
15. bit:位运算
16. str:字符串
17. array:数组
18. hash:哈希
